    What is BUT4Reuse?

    BUT4Reuse framework provides technologies for leveraging commonality and variability of software artefacts. Use it for analysing commonality and variability, feature identification, feature location, structural and semantic constraints discovery or for reusable assets construction

    Why BUT4Reuse?

    Easy-to-use state-of-the-art comparison techniques for software variants. Easily extensible for your own needs. BUT4Reuse is a framework that will help you in a wide range of situations: From providing helpful insights about how and where opportunistic reuse was performed in your set of software variants, to mining existing assets for Software Product Line adoption.
