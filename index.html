    What is BUT4Reuse?

    BUT4Reuse project provides technologies for analyzing commonality and variability of software artefacts. Use it for feature identification, feature localization or for structural and semantic constraints analysis between features.

    Why BUT4Reuse?

    Easy-to-use state-of-the-art comparison techniques for software variants. Easily extensible for your own needs. BUT4Reuse is the tool that will help you in a wide range of situations: From providing helpful insights about how and where opportunistic reuse was performed in your set of software variants, to mining existing assets for Software Product Line adoption.
